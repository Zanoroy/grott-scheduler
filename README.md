# Grott Scheduler

Professional scheduling system for Growatt inverter control via Grott.

## Features

- ✅ Web-based schedule management
- ✅ Single register & multiregister support
- ✅ Daily, Weekly, One-time schedules
- ✅ Conditional execution
- ✅ Pre-defined templates
- ✅ Custom curl commands
- ✅ Retry logic (up to 5 attempts)
- ✅ Pushover notifications
- ✅ Execution history
- ✅ Manual test execution

## Quick Start

```bash
cd /opt/grott-scheduler
./install.sh
```

Access web interface at: `http://your-server:5783`
